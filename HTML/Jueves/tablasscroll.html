Tablas Adaptables

Una tabla adaptable tiva mostrará una barra de desplazamiento horizontal si la pantalla es demasiado pequeño para mostrar el contenido completo. Cambie el tamaño de la ventana del navegador para ver el efecto:

Para crear una tabla adaptable, agregue un elemento contenedor (como div) con overflow-x: auto alrededor del elemento de la tabla:
